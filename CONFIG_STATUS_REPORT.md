## 📞 支持

### 故障排除
```bash
# 运行健康检查
~/.spacemacs.d/scripts/config-manager.sh health

# 修复包依赖
~/.spacemacs.d/scripts/config-manager.sh fix

# 查看配置状态
~/.spacemacs.d/scripts/config-manager.sh status
```

### 日志查看
- 错误日志: `logs/error.log`
- 审计日志: `logs/config-audit.log`
- 错误报告: `M-x my/generate-error-report`
